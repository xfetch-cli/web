# Plugins Index

xfetch provides a set of official plugins for additional functionality.

## Info Provider Plugins

- [docker](docker.md) — Docker container statistics
- [github-stats](github-stats.md) — GitHub user statistics
- [music-player](music-player.md) — Currently playing music (MPD / Spotify)
- [weather](weather.md) — Current weather conditions via wttr.in
- [timezone](timezone.md) — Timezone, date, and UTC offset
- [user-info](user-info.md) — User account information
- [display-resolution](display-resolution.md) — Monitor resolution and refresh rate
- [theme-detection](theme-detection.md) — Desktop theme detection (GTK / KDE)
- [theme-manager](theme-manager.md) — Theme registry browser and installer

## Logo Animation Plugins

- [animate-logo](animate-logo.md) — Animated ASCII logos with color effects
