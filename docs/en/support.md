# Support

Need help with installation, configuration, layouts, or runtime behavior of **xfetch**?

## Get Help

- **Issues:** [github.com/xfetch-cli/xfetch/issues](https://github.com/xfetch-cli/xfetch/issues)
- **Documentation:** [Getting Started](getting-started.md), [Configuration](configuration.md), [Layouts](layouts.md)
- **Security reports:** [x@xscriptor.com](mailto:x@xscriptor.com) (private disclosure)

## Before Opening an Issue

- Confirm your platform and version (Linux/macOS/Windows)
- Share your active config and layout file (remove sensitive details if any)
- Include command output and reproduction steps
- If visual output differs, attach screenshots when possible

## Response Expectations

Maintainers triage reports based on impact and reproducibility. Clear, minimal reproductions are resolved faster.
