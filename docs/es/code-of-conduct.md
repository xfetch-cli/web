# Código de Conducta

Se espera que los colaboradores y usuarios de xfetch colaboren de manera respetuosa, inclusiva y profesional.

## Nuestros Estándares

- Comuníquese con respeto y claridad técnica
- Proporcione comentarios constructivos sobre código, documentación y decisiones de diseño
- Dé la bienvenida a colaboradores de todas las plataformas y niveles de experiencia
- Mantenga las discusiones enfocadas en mejorar xfetch para todos

## Comportamiento Inaceptable

- Acoso, discriminación o lenguaje abusivo
- Ataques personales, burlas o escalada hostil
- Interrupción deliberada de discusiones o revisiones del proyecto
- Publicación de información privada sin permiso

## Cómo Reportar

Reporte incidentes a [x@xscriptor.com](mailto:x@xscriptor.com). Los reportes se tratan de forma confidencial.
