# Índice de Plugins

xfetch proporciona un conjunto de plugins oficiales para funcionalidad adicional.

## Plugins de Proveedor de Información

- [docker](docker.md) — Estadísticas de contenedores Docker
- [github-stats](github-stats.md) — Estadísticas de usuario de GitHub
- [music-player](music-player.md) — Música reproduciéndose actualmente (MPD / Spotify)
- [weather](weather.md) — Condiciones climáticas actuales vía wttr.in
- [timezone](timezone.md) — Zona horaria, fecha y offset UTC
- [user-info](user-info.md) — Información de cuenta de usuario
- [display-resolution](display-resolution.md) — Resolución de monitor y tasa de refresco
- [theme-detection](theme-detection.md) — Detección de tema de escritorio (GTK / KDE)
- [theme-manager](theme-manager.md) — Navegador e instalador de registros de temas

## Plugins de Animación de Logo

- [animate-logo](animate-logo.md) — Logos ASCII animados con efectos de color
