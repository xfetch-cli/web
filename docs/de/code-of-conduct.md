# Verhaltenskodex

Von Xfetch-Mitwirkenden und -Benutzern wird erwartet, dass sie auf respektvolle, integrative und professionelle Weise zusammenarbeiten.

## Unsere Standards

- Kommunizieren Sie mit Respekt und technischer Klarheit
- Geben Sie konstruktives Feedback zu Code, Dokumentation und Designentscheidungen
- Heißen Sie Mitwirkende aller Plattformen und Erfahrungsstufen willkommen
- Halten Sie Diskussionen auf die Verbesserung von xfetch fokussiert

## Inakzeptables Verhalten

- Belastigung, Diskriminierung oder beleidigende Sprache
- Personliche Angriffe, Verspottung oder feindselige Eskalation
- Vorsatzliche Storung von Projektdiskussionen oder Uberprufungen
- Veroffentlichung privater Informationen ohne Erlaubnis

## Melden

Melden Sie Vorfalle an [x@xscriptor.com](mailto:x@xscriptor.com). Meldungen werden vertraulich behandelt.
