# Plugin-Index

xfetch bietet eine Reihe offizieller Plugins für zusätzliche Funktionalität.

## Info-Provider-Plugins

- [docker](docker.md) — Docker-Container-Statistiken
- [github-stats](github-stats.md) — GitHub-Benutzerstatistiken
- [music-player](music-player.md) — Aktuelle Musikwiedergabe (MPD / Spotify)
- [weather](weather.md) — Aktuelle Wetterbedingungen via wttr.in
- [timezone](timezone.md) — Zeitzone, Datum und UTC-Offset
- [user-info](user-info.md) — Benutzerkontoinformationen
- [display-resolution](display-resolution.md) — Monitorauflösung und Bildwiederholrate
- [theme-detection](theme-detection.md) — Desktop-Theme-Erkennung (GTK / KDE)
- [theme-manager](theme-manager.md) — Theme-Registry-Browser und -Installierer

## Logo-Animation-Plugins

- [animate-logo](animate-logo.md) — Animierte ASCII-Logos mit Farbeffekten
